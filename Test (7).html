Examen de Práctica LPIC-1 (Test 7)

1. ¿Qué opción de mount permite montar un sistema de archivos solo en modo lectura?

  • A) -a
  • B) -o rw
  • C) -o ro
  • D) --readonly
Respuesta correcta: C. (-o ro u -o readonly).

2. ¿Qué hace el comando mount -a?

  • A) Monta todos los sistemas listados en /etc/fstab
  • B) Desmonta todas las particiones montadas
  • C) Muestra todos los sistemas montados
  • D) Sincroniza todos los sistemas de archivos
Respuesta correcta: A. (Monta todos los sistemas de archivos con la opción **auto**).

3. ¿Qué opción de mount permite especificar manualmente el tipo de sistema de archivos?

  • A) -F
  • B) -t
  • C) --fs-type
  • D) -f
Respuesta correcta: B. (-t o --types).

4. ¿Qué hace umount -l /mnt/data?

  • A) Desmonta el sistema inmediatamente
  • B) Desmonta de forma forzada
  • C) Desmonta de forma perezosa (lazy), posponiendo hasta que no esté en uso
  • D) Desmonta y borra el punto de montaje
Respuesta correcta: C.

5. El comando df -h muestra:

  • A) Uso de CPU y RAM en formato legible
  • B) Uso de disco en bloques de 512 bytes
  • C) Espacio en disco en formato legible por humanos
  • D) Solo los sistemas de archivos montados manualmente
Respuesta correcta: C. (df es para Disk Free, y -h es para formato legible por humanos).

6. ¿Qué hace df -i?

  • A) Muestra información de inodos usados y disponibles
  • B) Muestra tamaños en bytes
  • C) Muestra los discos conectados
  • D) Muestra el uso de disco del directorio actual
Respuesta correcta: A.

7. ¿Cuál es el significado del estado de un proceso R en ps o top?

  • A) Dormido (Sleeping)
  • B) Detenido (Stopped)
  • C) Ejecutándose (Running) o en cola para ejecución
  • D) Proceso zombie
Respuesta correcta: C.

8. ¿Qué significa el estado Z en un proceso?

  • A) Bloqueado (Waiting)
  • B) Detenido por señal (Stopped)
  • C) Proceso zombie (Terminado, pero su entrada en la tabla de procesos aún existe)
  • D) Proceso de suspensión (Sleeping)
Respuesta correcta: C.

9. El comando ps aux muestra:

  • A) Todos los procesos con formato de BSD (incluyendo usuarios de otros terminales)
  • B) Procesos con formato de System V
  • C) Solo procesos del usuario actual
  • D) Solo procesos que están ejecutándose
Respuesta correcta: A. (a: todos los procesos de todos los usuarios, u: muestra el proceso del usuario que lo invoca, x: incluye procesos sin terminal de control).

10. ¿Qué opción de ps muestra el árbol de procesos?

  • A) -A
  • B) -e
  • C) -f
  • D) -H
Respuesta correcta: D. (-H o --forest; también se puede usar pstree).

11. ¿Cuál es el PID (Process ID) de **init** (el primer proceso del sistema)?

  • A) 0
  • B) 1
  • C) El PID varía
  • D) El PID es siempre 1000
Respuesta correcta: B. (El primer proceso que arranca el kernel es el proceso **init** o **systemd** con PID 1).

12. En ps, ¿qué opción se utiliza para mostrar el identificador del padre del proceso (PPID)?

  • A) -f (formato completo)
  • B) -j (formato jobs)
  • C) -o ppid
  • D) Todas las anteriores
Respuesta correcta: D. (-f incluye PPID, -j incluye PPID, y -o ppid lo selecciona explícitamente).

13. ¿Qué comando se utiliza para mostrar continuamente el consumo de recursos de los procesos en tiempo real?

  • A) ps aux
  • B) top
  • C) htop
  • D) B y C son correctas
Respuesta correcta: D.

14. Dentro de top, ¿qué tecla se utiliza para cambiar el criterio de ordenación (ej. de CPU a Memoria)?

  • A) s
  • B) P
  • C) O
  • D) k
Respuesta correcta: O. (O para seleccionar el campo, P para ordenar por CPU, M para ordenar por memoria, k para matar un proceso).

15. En top, ¿qué tecla te permite matar un proceso?

  • A) q
  • B) k
  • C) x
  • D) r
Respuesta correcta: B.

16. ¿Qué comando se utiliza para enviar un proceso al segundo plano?

  • A) bg
  • B) fg
  • C) jobs
  • D) nice
Respuesta correcta: A. (bg para background; se usa en procesos suspendidos con Ctrl+Z).

17. ¿Qué comando se utiliza para traer un proceso en segundo plano al primer plano?

  • A) bg
  • B) fg
  • C) jobs
  • D) &
Respuesta correcta: B. (fg para foreground).

18. ¿Cómo se ejecuta un comando directamente en segundo plano?

  • A) comando &
  • B) bg comando
  • C) comando -b
  • D) comando -d
Respuesta correcta: A. (Se añade el símbolo ampersand **&** al final).

19. ¿Qué comando se utiliza para listar los trabajos (jobs) que se están ejecutando en segundo plano en el shell actual?

  • A) ps aux
  • B) jobs
  • C) bg -l
  • D) fg -l
Respuesta correcta: B.

20. ¿Qué hace el comando fg %2?

  • A) Ejecuta el trabajo 2 en segundo plano
  • B) Trae el trabajo 2 al primer plano
  • C) Muestra información del trabajo 2
  • D) Envía el trabajo actual al segundo plano
Respuesta correcta: B. (%N hace referencia al número de trabajo listado por jobs).

21. ¿Qué valor de **niceness** tiene la prioridad más alta (más favorable)?

  • A) 20
  • B) 0
  • C) -20
  • D) 10
Respuesta correcta: C. (El rango va de -20 a 19. Un valor más bajo significa mayor prioridad).

22. ¿Cuál es el valor por defecto de **niceness** para un proceso iniciado por un usuario normal?

  • A) 0
  • B) 10
  • C) 19
  • D) -10
Respuesta correcta: A.

23. ¿Qué comando se utiliza para ejecutar un programa con una prioridad de niceness de 15?

  • A) renice 15 comando
  • B) nice -n 15 comando
  • C) nice +15 comando
  • D) B y C son correctas
Respuesta correcta: B. (nice se usa para iniciar procesos con una prioridad, -n especifica el valor).

24. ¿Qué comando se utiliza para cambiar la prioridad de niceness de un proceso que ya está en ejecución?

  • A) nice
  • B) renice
  • C) priority
  • D) kill -PRIO
Respuesta correcta: B.

25. ¿Qué señal de kill indica una terminación elegante (solicitud de terminación) que permite al proceso guardar su trabajo?

  • A) SIGKILL (9)
  • B) SIGSTOP (19)
  • C) SIGTERM (15)
  • D) SIGHUP (1)
Respuesta correcta: C.

26. ¿Qué señal de kill no puede ser interceptada o ignorada por un proceso, asegurando su terminación inmediata?

  • A) SIGTERM (15)
  • B) SIGHUP (1)
  • C) SIGKILL (9)
  • D) SIGINT (2)
Respuesta correcta: C.

27. ¿Cómo se mata un proceso usando su PID (1234) de la manera más agresiva?

  • A) kill 1234
  • B) kill -9 1234
  • C) kill -SIGTERM 1234
  • D) kill -QUIT 1234
Respuesta correcta: B. (-9 es el número de la señal SIGKILL).

28. ¿Qué señal se utiliza comúnmente para indicarle a un demonio que recargue sus archivos de configuración sin terminar el proceso?

  • A) SIGTERM (15)
  • B) SIGHUP (1)
  • C) SIGKILL (9)
  • D) SIGINT (2)
Respuesta correcta: B. (SIGHUP, o Hang Up, es la señal estándar para recargar configuración).

29. ¿Qué comando se utiliza para encontrar el PID de un proceso por su nombre?

  • A) pidof
  • B) pgrep
  • C) ps -C
  • D) Todos los anteriores
Respuesta correcta: D. (pidof, pgrep y ps -C son formas válidas).

30. ¿Cómo se termina un proceso por su nombre (ej. **httpd**) en lugar de por su PID?

  • A) killall httpd
  • B) pkill httpd
  • C) kill $(pidof httpd)
  • D) Todos los anteriores
Respuesta correcta: D.

31. ¿Cuál es el comando correcto para apagar el sistema inmediatamente sin retraso?

  • A) shutdown now
  • B) init 0
  • C) halt
  • D) Todos los anteriores
Respuesta correcta: D. (shutdown now, init 0, systemctl poweroff, y halt logran un apagado inmediato).

32. ¿Qué comando se utiliza para reiniciar el sistema en 5 minutos y enviar un mensaje a los usuarios?

  • A) reboot 5 "Mensaje"
  • B) shutdown -r +5 "Mensaje"
  • C) init 6 +5 "Mensaje"
  • D) systemctl restart +5 "Mensaje"
Respuesta correcta: B. (-r es para reiniciar, +N es el retraso en minutos).

33. ¿Qué hace el comando shutdown -c?

  • A) Reinicia la computadora
  • B) Cancela un apagado o reinicio programado
  • C) Apaga de forma forzada
  • D) Muestra una cuenta regresiva
Respuesta correcta: B. (-c para cancel).

34. ¿Cuál es el nivel de ejecución (runlevel) que corresponde al apagado del sistema?

  • A) 6
  • B) 1
  • C) 0
  • D) 3
Respuesta correcta: C.

35. ¿Cuál es el nivel de ejecución (runlevel) que corresponde al reinicio del sistema?

  • A) 6
  • B) 0
  • C) 5
  • D) 3
Respuesta correcta: A.

36. ¿Qué comando se utiliza para cambiar al runlevel 3 (multi-user sin X) en SysV-init?

  • A) systemctl isolate multi-user.target
  • B) telinit 3
  • C) runlevel 3
  • D) init -r 3
Respuesta correcta: B. (telinit es el comando usado para cambiar el runlevel; en systemd es systemctl isolate).

37. ¿Qué comando te permite ver el runlevel anterior y el actual?

  • A) runlevel
  • B) who -r
  • C) systemctl get-default
  • D) A y B son correctas
Respuesta correcta: D.

38. ¿Cuál es el comando de **systemd** equivalente a init 6?

  • A) systemctl isolate reboot.target
  • B) systemctl reboot
  • C) systemctl start reboot
  • D) B y C son correctas
Respuesta correcta: B.

39. ¿Qué comando se utiliza para mostrar la información del kernel (nombre, versión, arquitectura)?

  • A) uname -a
  • B) kernelctl info
  • C) dmesg | grep kernel
  • D) cat /proc/version
Respuesta correcta: A.

40. ¿Qué opción de uname muestra la arquitectura del hardware?

  • A) -a
  • B) -r
  • C) -m
  • D) -n
Respuesta correcta: C. (-m o --machine).

41. ¿Qué archivo se utiliza para ver la información de la memoria RAM del sistema?

  • A) /proc/meminfo
  • B) /proc/cpuinfo
  • C) /etc/sysconfig/memory
  • D) /var/log/mem.log
Respuesta correcta: A. (También se puede usar free -h).

42. ¿Qué comando se utiliza para mostrar el uso de swap y memoria RAM en formato legible?

  • A) df -h
  • B) du -h
  • C) free -h
  • D) top -m
Respuesta correcta: C.

43. ¿Qué archivo contiene la información de la CPU (velocidad, cache, número de núcleos)?

  • A) /proc/meminfo
  • B) /proc/cpuinfo
  • C) /sys/devices/cpu/
  • D) /etc/default/cpu
Respuesta correcta: B.

44. ¿Qué comando se utiliza para mostrar los mensajes de arranque del kernel (Ring Buffer)?

  • A) journalctl -k
  • B) dmesg
  • C) A y B son correctas
  • D) cat /var/log/boot.log
Respuesta correcta: C.

45. ¿Qué comando se utiliza para mostrar la hora del sistema y ajustar la hora del hardware?

  • A) date
  • B) hwclock
  • C) timedatectl
  • D) B y C son correctas
Respuesta correcta: D. (hwclock es más granular; timedatectl es la herramienta de systemd más moderna).

46. ¿Qué hace timedatectl set-timezone Europe/Madrid?

  • A) Muestra la zona horaria actual
  • B) Muestra la lista de zonas horarias
  • C) Establece la zona horaria del sistema de forma persistente
  • D) Sincroniza la hora con un servidor NTP
Respuesta correcta: C.

47. ¿Qué comando se utiliza para montar un archivo ISO llamado **disk.iso** en **/mnt/cdrom**?

  • A) mount /dev/cdrom /mnt/cdrom
  • B) mount -o loop disk.iso /mnt/cdrom
  • C) mount -t iso9660 disk.iso /mnt/cdrom
  • D) B y C son correctas
Respuesta correcta: B. (-o loop es necesario para tratar un archivo como un dispositivo de bloques).

48. En el comando ps aux, ¿qué campo representa la prioridad del proceso?

  • A) PID
  • B) NI (Niceness)
  • C) STAT
  • D) TTY
Respuesta correcta: B. (NI es Niceness, que se usa para calcular la prioridad).

49. ¿Qué comando se utiliza para mostrar el estado de los servicios de un usuario (no del sistema)?

  • A) systemctl --user status
  • B) systemctl status
  • C) service --user
  • D) systemctl --runtime
Respuesta correcta: A.

50. ¿Qué hace systemctl get-default?

  • A) Muestra el runlevel por defecto (default target)
  • B) Inicia el servicio por defecto
  • C) Muestra todos los servicios activos
  • D) Configura el runlevel por defecto
Respuesta correcta: A.

51. ¿Cuál es el significado del bit **Sticky** (valor 1000) en un directorio, como /tmp?

  • A) El directorio no se puede borrar
  • B) Solo el propietario o root pueden eliminar archivos en ese directorio
  • C) Todos los archivos creados heredan el grupo
  • D) El directorio se mantiene en caché
Respuesta correcta: B.

52. Si un proceso llamado **sleep** se está ejecutando en segundo plano, ¿con qué comando usarías para matarlo?

  • A) killall sleep
  • B) pkill sleep
  • C) kill %1
  • D) Cualquiera de las anteriores
Respuesta correcta: D. (Siempre que %1 sea el ID de trabajo correcto. killall y pkill terminan procesos por nombre).

53. ¿Qué hace kill -HUP 1234?

  • A) Termina el proceso inmediatamente
  • B) Reinicia el proceso (HangUp)
  • C) Lo suspende
  • D) Cambia su prioridad
Respuesta correcta: B. (Dependiendo del proceso, SIGHUP puede recargar o reiniciar).

54. ¿Qué comando muestra los procesos con su prioridad y uso de CPU ordenados descendente por uso?

  • A) ps aux --sort=-%cpu
  • B) ps aux --sort=%cpu
  • C) top -a
  • D) ps -eo cpu,pri,pid
Respuesta correcta: A. (- indica orden descendente).

55. ¿Cuál comando cambia la prioridad de un proceso en ejecución a valor 10 de niceness?

  • A) renice 10 -p PID
  • B) nice -n 10 PID
  • C) renice -n PID 10
  • D) nice PID 10
Respuesta correcta: A. (renice requiere el valor primero, luego -p PID o -u user).

56. ¿Qué hace renice +5 -u usuario?

  • A) Aumenta la prioridad de todos los procesos del usuario
  • B) Reduce la prioridad de todos los procesos del usuario
  • C) Mata los procesos del usuario
  • D) Suspende los procesos del usuario
Respuesta correcta: B. (Aumentar el valor de niceness (ej. de 0 a 5) **reduce** la prioridad real).

57. ¿Qué ocurre si ejecutas nice -n -5 comando sin ser root?

  • A) Se ejecuta con prioridad más alta
  • B) El comando falla
  • C) Se ignora el valor negativo y se usa el por defecto
  • D) Se ejecuta con prioridad más baja
Respuesta correcta: C. (Solo root puede usar valores negativos de niceness).
