Examen de Práctica LPIC-1 (Test 14)

1.- ¿Qué significa CLI?
Command Line Interface (Interfaz de Línea de Comandos)
Command Learning Interface (Interfaz de Aprendizaje de Comandos)
Core Line Interface (Interfaz de Línea Esencial)
Core Learning Interface (Interfaz de Aprendizaje Esencial)

Respuesta Correcta: Command Line Interface (Interfaz de Línea de Comandos)

Explicación: **CLI** significa **Command Line Interface**, que es una interfaz de texto utilizada para operar software y sistemas operativos.

2.- Los comandos distinguen entre mayúsculas y minúsculas. ¿Verdadero o Falso?
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: El sistema operativo Linux es **sensible a mayúsculas y minúsculas** (case-sensitive).

3.- De las siguientes opciones, ¿Cuál NO es una manera correcta de combinar las opciones?
-I-r
-rl
-rl
-Ir

Respuesta Correcta: -I-r

Explicación: La forma estándar es agrupar opciones después de un solo guion (ej: `-rl` o `-Ir`). Usar un segundo guion como en **`-I-r`** es incorrecto.

4.- ¿Qué comando te permite cambiar de directorio?
chdir
ch
Is
cd

Respuesta Correcta: cd

Explicación: El comando **`cd`** (change directory) se utiliza para cambiar el directorio de trabajo actual.

5.- El carácter ~ (tilde) se usa para representar:
El directorio de inicio de un usuario
Nada, no tiene un significado especial
El directorio actual
El directorio sobre el directorio actual

Respuesta Correcta: El directorio de inicio de un usuario

Explicación: El carácter **`~`** (tilde) es un acceso directo al directorio **home** del usuario logueado.

6.- ¿Qué opción para Is ordenará la salida por tamaño de archivo?
-Z
-r
-S
-s

Respuesta Correcta: -S

Explicación: La opción **`-S`** del comando `ls` ordena la lista de archivos por **tamaño** de archivo (Size), de mayor a menor.

7.- Los caracteres .. (doble punto) se usan para representar:
Nada, no tiene un significado especial
El directorio actual
El directorio "home" de un usuario
El directorio arriba del directorio actual

Respuesta Correcta: El directorio arriba del directorio actual

Explicación: Los caracteres **`..`** representan el **directorio padre** o el directorio inmediatamente superior.

8.- El carácter . (punto) se usa para representar:
Nada, no tiene un significado especial
El directorio actual
El directorio arriba del directorio actual
El directorio "home" de un usuario

Respuesta Correcta: El directorio actual

Explicación: El carácter **`.`** (punto) representa el **directorio de trabajo actual**.

9.- El primer carácter en una lista larga, resultado de ls -l indica:
Permisos
Recuento de Enlaces Físicos
Tipo de Archivo
Tamaño de Archivo

Respuesta Correcta: Tipo de Archivo

Explicación: En la salida de `ls -l`, el primer carácter (ej: `d` para directorio, `-` para archivo) indica el **tipo de archivo**.

10.- ¿Cuál de los siguientes comandos creará un nuevo shell conectado como el usuario root?
sudo
su
cp
dd

Respuesta Correcta: su

Explicación: El comando **`su`** (substitute user), ejecutado sin argumentos, cambia al usuario **root** e inicia un nuevo shell interactivo.

11.- ¿Qué comando se usa para copiar archivos a nivel de bit?
cp
dd
...
...

Respuesta Correcta: dd

Explicación: El comando **`dd`** se utiliza para copiar datos sin procesar, como particiones de disco o información a nivel de bloque/bit.

12.- ¿Qué comando le permite al usuario cambiar los permisos de un archivo?
perm
chperm
chmod
chown

Respuesta Correcta: chmod

Explicación: El comando **`chmod`** (change mode) se usa para cambiar los **permisos** de acceso a archivos y directorios.

13.- ¿Cuál de los siguientes comandos regresará solo líneas que comiencen con test?
grep '^test' file.txt
grep 'test' file.txt
grep 'test\$' file.txt
grep '\$test' file.txt

Respuesta Correcta: grep '^test' file.txt

Explicación: En `grep`, el carácter **`^`** (circunflejo) ancla el patrón al **inicio de la línea**.

14.- ¿Cuál de los siguientes comandos se usa para cambiar la propiedad de un archivo?
chperm
chown
chmod
chow

Respuesta Correcta: chown

Explicación: El comando **`chown`** (change owner) se usa para cambiar el **propietario** (usuario) y/o el grupo de un archivo.

15.- Cambiar el usuario propietario de un archivo requiere acceso administrativo. ¿Verdadero o Falso?
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Solo el usuario **root** (o un usuario con privilegios `sudo`) puede cambiar la propiedad de un archivo a **otro usuario**.

16.- ¿Cuál de los siguientes comandos puede usarse para renombrar un archivo?
name
cp
rn
mv

Respuesta Correcta: mv

Explicación: El comando **`mv`** (move) se utiliza para **mover** un archivo o para **renombrarlo**.

17.- El comando mv requiere al menos dos argumentos. ¿Verdadero o Falso?
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: `mv` requiere al menos el **origen** (archivo a mover) y el **destino** (nuevo nombre o ubicación).

18.- ¿Qué comando se puede utilizar para copiar un archivo?
mv
cp
cy
rm

Respuesta Correcta: cp

Explicación: El comando **`cp`** (copy) se utiliza para hacer copias de archivos y directorios.

19.- El comando cp requiere al menos dos argumentos. ¿Verdadero o Falso?
Falso
Verdadero

Respuesta Correcta: Verdadero

Explicación: `cp` requiere el **origen** (archivo a copiar) y el **destino** (ubicación de la copia).

20.- ¿Qué opción del comando rm permitirá a los usuarios borrar directorios?
-1
-r
-a
-d

Respuesta Correcta: -r

Explicación: La opción **`-r`** (recursive) es necesaria para eliminar **directorios** y su contenido con `rm`.

21.- ¿Cuál de los siguientes comandos se utiliza para filtrar texto?
dd
regex
grep
text

Respuesta Correcta: grep

Explicación: El comando **`grep`** (Global Regular Expression Print) se utiliza para buscar líneas de texto que coincidan con un patrón.

22.- ¿Cuál de los siguientes comandos regresará solo líneas que terminan con test?
grep 'test' file.txt
grep 'test' file.txt
grep 'test\$' file.txt
grep '\$test' file.txt

Respuesta Correcta: grep 'test\$' file.txt

Explicación: En `grep`, el carácter **`$`** (dólar) ancla el patrón al **final de la línea**.

23.- ¿Cuál de las siguientes líneas no sera devuelta por el comando grep '^0-9' file.txt?
3121991
My favorite food is avocados.
Hello my name is Joe.
I am 37 years old.

Respuesta Correcta: 3121991

Explicación: Asumiendo el patrón literal, `grep '^0-9'` busca líneas que comiencen con "0-9". La línea **`3121991`** comienza con "3", por lo que no coincide y no sería devuelta.

24.- ¿Cuál de las siguientes líneas será entregada por el comando grep 'b[oe]t' file.txt?
beet
bet
boet
boot

Respuesta Correcta: bet

Explicación: El patrón **`b[oe]t`** coincide con `b` seguido de una `o` **o** una `e`, seguido de `t`. Las coincidencias válidas son `bot` y **`bet`**.

25.- ¿Cuál de los siguientes comandos puede ser usado para mostrar la información de configuración de red?
ifconfig
netconfig
pwd
net

Respuesta Correcta: ifconfig

Explicación: El comando **`ifconfig`** (interface configuration) se utiliza tradicionalmente para ver y configurar las interfaces de red del sistema.

26.- ¿Cuál de los sigueintes comandos muestra todos los procesos?
ps
ps --all
ps-a
ps -e

Respuesta Correcta: ps -e

Explicación: La opción **`-e`** (o `-a` en algunos sistemas) del comando `ps` muestra **todos** los procesos en ejecución en el sistema.

27.- ¿Cuál de los siguientes comandos borrará todos los archivos de un paquete?
apt-get delete
apt-get trash
apt-get remove
apt-get purge

Respuesta Correcta: apt-get purge

Explicación: **`apt-get purge`** elimina el paquete *y* sus archivos de configuración. `remove` solo elimina el paquete, manteniendo la configuración.

28.- Los usuarios pueden actualizar sus propias contraseñas. ¿Verdadero o Falso?

Verdadero

30.- ¿Qué opción se puede utilizar para ver información sobre el estado de la contraseña del usuario actual?

shutdown

shutdown +0

ps --all
ps-e
ps
ps -a

Respuesta Correcta: ps --all / ps -e

Explicación: La opción **`ps -e`** o **`ps --all`** (dependiendo de la sintaxis del sistema) se utilizan para listar todos los procesos.

33.- ¿Cuál de los siguientes comandos actualizará todos los paquetes?

Respuesta Correcta: apt-get upgrade

Explicación: **`apt-get upgrade`** instala las versiones más nuevas de todos los paquetes instalados en el sistema.

34.- ¿Cuál de los siguientes comandos debe ejecutarse antes de instalar un paquete?
apt-get upgrade
apt-get update
apt-get install
apt-get search

Respuesta Correcta: apt-get update

Explicación: **`apt-get update`** descarga la información más reciente de los repositorios para que el sistema sepa qué paquetes están disponibles.

35.- El usuario root puede cambiar la contraseña de cualquier usuario. ¿Verdadero o falso?

Respuesta Correcta: Verdadero

Explicación: El usuario **root** tiene privilegios para cambiar la contraseña de cualquier usuario. (Repetida de la Q29)

38.- Los comandos distinguen entre mayúsculas y minúsculas. ¿Verdadero o falso?

39.- Uno de los trabajos del kernel es:

41.- El código fuente se refiere a:

67.- ¿Qué variable de entorno contiene una lista de directorios en los que se buscan comandos para ejecutar?

PS1

68.- Seleccione el comando que puede informar la ubicación de un comando:
where
which
what

Respuesta Correcta: which

Explicación: El comando **`which`** busca en los directorios listados en `$PATH` y muestra la ruta completa al comando que se ejecutará.

69.- Un par de comillas simples (') evitará que el shell interprete cualquier metacarácter. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Las comillas **simples (`'`)** en el shell se utilizan para la **cita fuerte**, lo que significa que eliminan el significado especial de todos los metacaracteres.

70.- Un par de comillas dobles (") evitará que el shell interprete cualquier metacarácter. Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Las comillas **dobles (`"`)** permiten la **expansión de variables** (ej. `$VAR`) y la **sustitución de comandos** (ej. `` o `$(...)`), por lo que no evitan la interpretación de *todos* los metacaracteres.

Verdadero
