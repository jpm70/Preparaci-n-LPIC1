Examen de Práctica LPIC-1 (Test 8) - Comandos de Red

1. ¿Qué comando muestra la tabla de enrutamiento del kernel?

  • A) netstat -n
  • B) route -n
  • C) ip addr
  • D) ifconfig
Respuesta correcta: B. (route -n)

2. ¿Qué hace route add default gw 192.168.1.1?

  • A) Añade una nueva red local
  • B) Define la puerta de enlace predeterminada del sistema
  • C) Cambia la dirección IP del equipo
  • D) Configura un túnel
Respuesta correcta: B. (Define la puerta de enlace predeterminada del sistema)

3. ¿Qué opción de route elimina una ruta específica?

  • A) route del
  • B) route -r
  • C) route rm
  • D) route remove
Respuesta correcta: A. (route del)

4. ¿Qué muestra el comando arp -n?

  • A) El número de paquetes recibidos
  • B) La tabla ARP sin resolver nombres de host
  • C) Direcciones IP activas con nombres DNS
  • D) La tabla de rutas
Respuesta correcta: B. (La tabla ARP sin resolver nombres de host)

5. ¿Cómo se elimina manualmente una entrada ARP específica?

  • A) arp -a 192.168.0.5
  • B) arp -d 192.168.0.5
  • C) arp --remove 192.168.0.5
  • D) ip neigh del 192.168.0.5
Respuesta correcta: B. (arp -d 192.168.0.5)

6. ¿Qué hace tcpdump -i eth0 port 80?

  • A) Captura tráfico UDP en el puerto 80
  • B) Captura tráfico TCP en el puerto 80
  • C) Captura cualquier tráfico de red en eth0
  • D) Captura tráfico de salida únicamente
Respuesta correcta: B. (Captura tráfico TCP en el puerto 80)

7. ¿Qué opción de tcpdump muestra el contenido en formato ASCII?

  • A) -X
  • B) -A
  • C) -v
  • D) -x
Respuesta correcta: B. (-A)

8. ¿Cómo se limita la captura de tcpdump a solo 100 paquetes?

  • A) tcpdump -n 100
  • B) tcpdump -c 100
  • C) tcpdump -p 100
  • D) tcpdump --limit 100
Respuesta correcta: B. (tcpdump -c 100)

9. ¿Qué hace tcpdump -w captura.pcap?

  • A) Muestra los paquetes en pantalla
  • B) Guarda los paquetes en un archivo sin mostrarlos
  • C) Filtra paquetes de escritura
  • D) Muestra los encabezados TCP
Respuesta correcta: B. (Guarda los paquetes en un archivo sin mostrarlos)

10. El comando whois openai.com obtiene:

  • A) El propietario y datos del dominio
  • B) Los registros DNS del dominio
  • C) El TTL del dominio
  • D) La dirección IP del servidor
Respuesta correcta: A. (El propietario y datos del dominio)

11. ¿Qué campo de whois suele mostrar la organización propietaria del dominio?

  • A) Admin Email
  • B) Registrant Organization
  • C) Owner Name
  • D) NetName
Respuesta correcta: B. (Registrant Organization)

12. ¿Qué hace mtr 8.8.8.8?

  • A) Muestra los registros DNS
  • B) Hace trazado dinámico con estadísticas de pérdida y latencia
  • C) Lanza pings simultáneos a todos los saltos
  • D) Calcula el tiempo de respuesta promedio
Respuesta correcta: B. (Hace trazado dinámico con estadísticas de pérdida y latencia)

13. En mtr, ¿qué tecla muestra la vista “report mode”?

  • A) r
  • B) d
  • C) e
  • D) m
Respuesta correcta: A. (r)

14. vnstat obtiene sus datos de:

  • A) /proc/net/dev
  • B) /etc/network/interfaces
  • C) /var/log/syslog
  • D) /proc/net/tcp
Respuesta correcta: A. (/proc/net/dev)

15. ¿Qué hace vnstat -l?

  • A) Muestra estadísticas diarias
  • B) Monitorea tráfico en tiempo real
  • C) Genera un log en segundo plano
  • D) Resetea contadores
Respuesta correcta: B. (Monitorea tráfico en tiempo real)

16. ¿Qué muestra ifconfig eth0?

  • A) Configuración de red global
  • B) Solo la configuración de la interfaz eth0
  • C) Las rutas de red
  • D) Los sockets abiertos
Respuesta correcta: B. (Solo la configuración de la interfaz eth0)

17. ¿Qué hace ifconfig eth0 down?

  • A) Deshabilita la interfaz eth0
  • B) Reinicia la interfaz
  • C) Borra la configuración IP
  • D) Elimina la ruta predeterminada
Respuesta correcta: A. (Deshabilita la interfaz eth0)

18. ¿Qué comando reemplaza a ifconfig en sistemas modernos?

  • A) netstat
  • B) ip addr
  • C) ipconfig
  • D) route
Respuesta correcta: B. (ip addr)

19. ¿Qué devuelve hostname -I?

  • A) El nombre del host completo
  • B) Las direcciones IP asignadas al host
  • C) El FQDN del sistema
  • D) El nombre NetBIOS
Respuesta correcta: B. (Las direcciones IP asignadas al host)

20. ¿Cómo se cambia el nombre de host temporalmente?

  • A) hostnamectl set-hostname temporal
  • B) hostname temporal
  • C) Editando /etc/hostname
  • D) Reiniciando el servicio network
Respuesta correcta: B. (hostname temporal)

21. ¿Qué protocolo usa ping para probar conectividad?

  • A) UDP
  • B) ICMP
  • C) TCP
  • D) ARP
Respuesta correcta: B. (ICMP)

22. ¿Cómo se limita ping a 5 paquetes enviados?

  • A) ping -l 5
  • B) ping -n 5
  • C) ping -c 5
  • D) ping --limit 5
Respuesta correcta: C. (ping -c 5)

23. ¿Qué comando muestra la ruta seguida por los paquetes hasta un destino?

  • A) ping
  • B) trace
  • C) traceroute
  • D) netstat
Respuesta correcta: C. (traceroute)

24. ¿Qué opción de traceroute usa ICMP en lugar de UDP?

  • A) -I
  • B) -T
  • C) -U
  • D) -i
Respuesta correcta: A. (-I)

25. ¿Qué comando muestra las direcciones IP y máscaras configuradas en todas las interfaces?

  • A) ip route
  • B) ip addr show
  • C) ip link
  • D) ip netns
Respuesta correcta: B. (ip addr show)

26. ¿Cómo se agrega una ruta persistente con el comando ip?

  • A) ip route add 10.0.0.0/24 via 192.168.1.1
  • B) route add 10.0.0.0 mask 255.255.255.0
  • C) ip add route 10.0.0.0/24 dev eth0
  • D) ipconfig route add
Respuesta correcta: A. (ip route add 10.0.0.0/24 via 192.168.1.1)

27. ¿Qué hace ip link set eth0 up?

  • A) Reinicia la conexión
  • B) Habilita la interfaz
  • C) Cambia el modo promiscuo
  • D) Cambia la IP
Respuesta correcta: B. (Habilita la interfaz)

28. ¿Qué comando muestra las conexiones TCP abiertas?

  • A) netstat -t
  • B) netstat -u
  • C) netstat -a
  • D) netstat -n
Respuesta correcta: A. (netstat -t)

29. ¿Qué opción de netstat muestra los procesos asociados a los puertos?

  • A) -n
  • B) -p
  • C) -l
  • D) -s
Respuesta correcta: B. (-p)

30. ¿Qué hace ss -lntp?

  • A) Muestra conexiones UDP activas
  • B) Muestra sockets TCP en escucha con PIDs
  • C) Lista interfaces activas
  • D) Muestra uso de memoria por conexión
Respuesta correcta: B. (Muestra sockets TCP en escucha con PIDs)

31. ¿Qué diferencia hay entre ss y netstat?

  • A) ss es más moderno y rápido que netstat
  • B) netstat soporta IPv6 y ss no
  • C) ss no muestra PIDs
  • D) netstat usa sockets UDP únicamente
Respuesta correcta: A. (ss es más moderno y rápido que netstat)

32. ¿Qué comando se usa para resolver nombres a direcciones IP de forma interactiva?

  • A) dig
  • B) nslookup
  • C) whois
  • D) traceroute
Respuesta correcta: B. (nslookup)

33. ¿Qué hace nslookup -type=MX gmail.com?

  • A) Muestra los registros de correo (MX) del dominio
  • B) Muestra el TTL
  • C) Muestra los servidores DNS
  • D) Devuelve solo la IP principal
Respuesta correcta: A. (Muestra los registros de correo (MX) del dominio)

34. ¿Qué comando de dig devuelve solo la IP de un dominio?

  • A) dig +short dominio.com
  • B) dig -ip dominio.com
  • C) dig -a dominio.com
  • D) dig +only dominio.com
Respuesta correcta: A. (dig +short dominio.com)

35. ¿Cómo se consulta el servidor DNS 1.1.1.1 con dig?

  • B) dig -s 1.1.1.1 dominio.com
  • C) dig -d 1.1.1.1 dominio.com
  • D) dig 1.1.1.1 dominio.com

  • B) Descarga recursivamente todo el sitio

  • D) Muestra el contenido

37. ¿Qué hace wget -c archivo.iso?

  • B) Continúa una descarga interrumpida

38. ¿Qué opción de wget evita seguir redirecciones HTTP?

  • A) --no-check-certificate

41. ¿Qué opción de curl permite seguir redirecciones automáticamente?

  • A) -L
  • B) -r
  • C) --auto
  • D) -F
Respuesta correcta: A. (-L)

42. ¿Qué diferencia hay entre curl -d y curl -F?

  • A) -d envía datos como JSON, -F como formulario multipart
  • B) -F es solo para GET
  • C) -d envía cabeceras, -F contenido binario

43. ¿Qué opción de curl permite definir una cabecera HTTP personalizada?

  • A) -H
  • B) -C
  • C) -D
  • D) --header-set
Respuesta correcta: A. (-H)
