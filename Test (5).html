Examen de Práctica LPIC-1 (Test 93) - Archivos, Compresión y Navegación

1. ¿Qué comando crea un archivo tar llamado backup.tar con los contenidos del directorio /home/user?

  • A) tar -cvf backup.tar /home/user
  • B) tar -czf backup.tar /home/user
  • C) tar -cf /home/user backup.tar
  • D) tar -cv /home/user backup.tar
Respuesta correcta: A. (c: crear, v: detallado, f: nombre de archivo)

2. En tar, ¿qué opción permite comprimir usando gzip directamente?

  • A) -z
  • B) -j
  • C) -J
  • D) -Z
Respuesta correcta: A. (La opción -z usa gzip para crear un .tar.gz)

3. ¿Qué opción de tar permite verificar la integridad de un archivo .tar sin extraerlo?

  • A) -tvf
  • B) -tf
  • C) -Wf
  • D) --verify
Respuesta correcta: D.

4. ¿Cómo se extrae un archivo backup.tar.gz en un solo paso?

  • A) tar -xvf backup.tar.gz
  • B) tar -zxvf backup.tar.gz
  • C) gzip -d backup.tar.gz
  • D) tar -zcvf backup.tar.gz
Respuesta correcta: B. (x: extraer, z: gzip, v: detallado, f: archivo)

5. ¿Qué hace la opción -t en tar?

  • A) Testea el archivo
  • B) Lista el contenido del tar
  • C) Trunca el archivo
  • D) Transfiere a otro directorio
Respuesta correcta: B.

6. ¿Qué opción de tar se utiliza para excluir archivos durante la creación del tar?

  • A) --skip
  • B) --exclude
  • C) -X
  • D) --omit
Respuesta correcta: B.

7. ¿Qué hace el comando bzip2 -d archivo.bz2?

  • A) Comprime el archivo con bzip2
  • B) Descomprime el archivo con bzip2
  • C) Muestra el contenido comprimido
  • D) Borra el archivo original
Respuesta correcta: B. (-d es la opción de descompresión o decompress)

8. ¿Qué comando se usa para comprimir archivos con compresión bzip2 en tar?

  • A) -z
  • B) -j
  • C) -J
  • D) -Z
Respuesta correcta: B.

9. ¿Cuál es el comando para comprimir un archivo con el máximo nivel de compresión usando gzip?

  • A) gzip -best archivo
  • B) gzip -9 archivo
  • C) gzip -max archivo
  • D) gzip -v archivo
Respuesta correcta: B. (El nivel 9 es el máximo en gzip)

10. ¿Qué comando se utiliza para listar el contenido de un archivo .zip sin extraerlo?

  • A) zip -l archivo.zip
  • B) unzip -l archivo.zip
  • C) tar -tf archivo.zip
  • D) cat archivo.zip
Respuesta correcta: B.

11. ¿Cuál es el comando para cambiar el directorio actual al directorio home del usuario?

  • A) cd ~
  • B) cd $HOME
  • C) cd
  • D) Todos los anteriores
Respuesta correcta: D. (cd por sí solo, cd ~ y cd $HOME tienen el mismo efecto)

12. ¿Qué hace el comando cd -?

  • A) Regresa al directorio raíz
  • B) Regresa al directorio home
  • C) Regresa al directorio anterior
  • D) Muestra la lista de directorios recientes
Respuesta correcta: C.

13. ¿Qué variable de entorno almacena la ruta del directorio anterior?

  • A) $PWD
  • B) $OLDPWD
  • C) $CDPATH
  • D) $HOME
Respuesta correcta: B.

14. ¿Qué comando muestra la lista de directorios apilados (pushd/popd)?

  • A) dirs
  • B) cd -l
  • C) showdirs
  • D) lsdirs
Respuesta correcta: A.

15. ¿Qué comando agrega el directorio /etc a la pila de directorios?

  • A) pushd /etc
  • B) dirs -a /etc
  • C) cdstack /etc
  • D) push /etc
Respuesta correcta: A.

16. ¿Qué comando elimina el directorio actual de la pila y cambia al directorio anterior?

  • A) popd
  • B) dirs -r
  • C) cd -
  • D) popstack
Respuesta correcta: A.

17. ¿Qué comando usarías para ver todos los archivos en un directorio (incluidos los ocultos) en formato de lista larga y legible para humanos?

  • A) ls -al
  • B) ls -alh
  • C) ls -aR
  • D) ls -lH
Respuesta correcta: B. (-a: todos, -l: lista larga, -h: legible para humanos)

18. ¿Qué opción de cp sobrescribe los archivos de destino solo si son más antiguos que el archivo de origen?

  • A) -i
  • B) -n
  • C) -u
  • D) -f
Respuesta correcta: C. (-u o --update)

19. ¿Qué opción de rm elimina directorios y su contenido de forma recursiva?

  • A) -d
  • B) -r
  • C) -R
  • D) B y C son correctas
Respuesta correcta: D. (-r y -R son sinónimos para recursivo)

20. ¿Qué hace el comando mv archivo1 archivo2?

  • A) Mueve archivo1 al directorio archivo2
  • B) Copia archivo1 a archivo2
  • C) Renombra archivo1 a archivo2
  • D) Copia y renombra archivo1 a archivo2
Respuesta correcta: C. (Si el destino es un archivo, lo renombra)

21. ¿Qué comando se utiliza para crear un enlace simbólico (soft link) a /etc/hosts llamado myhosts en el directorio actual?

  • A) ln /etc/hosts myhosts
  • B) ln -s /etc/hosts myhosts
  • C) link -s /etc/hosts myhosts
  • D) ln -h /etc/hosts myhosts
Respuesta correcta: B. (-s es para enlace simbólico)

22. ¿Qué sucede si el archivo de origen de un enlace duro se elimina?

  • A) El enlace duro se rompe
  • B) El archivo en el disco permanece accesible a través del enlace duro
  • C) El enlace duro se convierte en un enlace simbólico
  • D) El sistema operativo avisa de un error
Respuesta correcta: B. (Solo cuando el contador de enlaces llega a cero, el archivo se elimina)

23. ¿Qué opción del comando ls muestra el inodo de un archivo?

  • A) -l
  • B) -i
  • C) -n
  • D) -a
Respuesta correcta: B.

24. ¿Qué hace el comando mkdir -p a/b/c?

  • A) Crea el directorio a/b/c, pero solo si a y b ya existen
  • B) Crea el directorio a/b/c y cualquier directorio padre que no exista
  • C) Crea los directorios a, b y c en paralelo
  • D) Muestra un error si a/b/c ya existe
Respuesta correcta: B. (-p o --parents)

25. ¿Qué comando se utiliza para crear un archivo vacío con una marca de tiempo específica?

  • A) touch -t AAMMDDhhmm archivo
  • B) touch -d "fecha" archivo
  • C) settime archivo
  • D) A y B son correctas
Respuesta correcta: D.

26. ¿Qué hace el comando file archivo.bin?

  • A) Abre el archivo en un editor de texto
  • B) Muestra el contenido del archivo
  • C) Muestra el tipo de archivo (ej. ASCII text, JPEG image, etc.)
  • D) Muestra los permisos del archivo
Respuesta correcta: C.

27. ¿Qué comando se utiliza para mostrar el uso de disco del directorio actual, legible para humanos y con un resumen total?

  • A) df -h
  • B) du -sh
  • C) ls -lh
  • D) du -ah
Respuesta correcta: B. (du para uso de disco, -s para resumen, -h para formato legible)

28. ¿Qué opción de du evita cruzar los límites del sistema de archivos al calcular el uso de disco?

  • A) -x
  • B) -l
  • C) -a
  • D) -s
Respuesta correcta: A. (-x o --one-file-system)

29. ¿Qué comando se utiliza para buscar el archivo de la página man para el comando tar?

  • A) locate tar
  • B) whereis tar
  • C) which tar
  • D) find / -name tar
Respuesta correcta: B. (whereis localiza binarios, fuentes y páginas man)

30. ¿Qué hace el comando updatedb?

  • A) Actualiza los paquetes del sistema
  • B) Actualiza la base de datos de man pages
  • C) Actualiza la base de datos de locate
  • D) Actualiza la tabla de enrutamiento
Respuesta correcta: C.

31. ¿Qué comando se utiliza para buscar archivos modificados hace más de 30 días en el directorio actual?

  • A) find . -mtime +30
  • B) find . -ctime +30
  • C) find . -atime +30
  • D) find . -mmin +30
Respuesta correcta: A.

32. ¿Qué opción del comando find se utiliza para buscar solo directorios?

  • A) -type f
  • B) -type d
  • C) -dir
  • D) -type l
Respuesta correcta: B. (-type d)

33. ¿Qué hace el comando find /home -user juan?

  • A) Busca archivos y directorios con el nombre "juan"
  • B) Busca archivos y directorios propiedad del usuario "juan"
  • C) Busca archivos y directorios que contengan el texto "juan"
  • D) Busca archivos modificados por "juan"
Respuesta correcta: B.

34. ¿Qué comando ejecuta rm -f {} en cada archivo encontrado por find?

  • A) find ... | exec rm -f
  • B) find ... -exec rm -f {} \;
  • C) find ... -xargs rm -f
  • D) B y C son correctas
Respuesta correcta: B.

35. ¿Qué comando se utiliza para mostrar solo los archivos binarios que se encuentran en el PATH del sistema?

  • A) whereis -b
  • B) which -b
  • C) type -a
  • D) ls -F | grep *
Respuesta correcta: B. (which es específico para ejecutables en el PATH)

36. ¿Qué hace find . -type f -size +1M?

  • A) Busca archivos que tienen exactamente 1MB
  • B) Busca archivos que tienen más de 1MB
  • C) Busca directorios que tienen más de 1MB
  • D) Busca archivos que tienen menos de 1MB
Respuesta correcta: B. (+1M indica "más grande que 1 Megabyte")

37. ¿Qué opción de gzip evita que el comando borre el archivo original después de comprimir?

  • A) -c
  • B) -k
  • C) -v
  • D) -f
Respuesta correcta: B. (-k o --keep)

38. ¿Qué comando se utiliza para buscar todas las líneas que contienen la palabra "error" en todos los archivos .log dentro del directorio /var/log de forma recursiva?

  • A) grep error /var/log/*.log
  • B) grep -r error /var/log/
  • C) find /var/log/ -name "*.log" | grep error
  • D) locate error /var/log/
Respuesta correcta: B. (-r para recursivo)

39. ¿Qué comando se utiliza para mostrar los permisos octales de un archivo?

  • A) ls -l
  • B) stat -c %a archivo
  • C) stat -c %p archivo
  • D) chmod -n archivo
Respuesta correcta: B.

40. ¿Cuál es el significado del octal 0777 para un archivo?

  • A) Solo lectura para todos
  • B) Lectura, escritura, ejecución para propietario, grupo y otros
  • C) Solo el propietario tiene todos los permisos
  • D) Solo el grupo tiene todos los permisos

41. ¿Cuál es el valor predeterminado de umask en la mayoría de los sistemas Linux?

  • A) 0000
  • B) 0022
  • C) 0777
  • D) 0002
Respuesta correcta: B.

42. Si umask es 027, ¿cuáles serán los permisos de un nuevo archivo creado?

  • A) 640
  • B) 750
  • C) 660
  • D) 770
Respuesta correcta: A. (Permisos base para archivo 666 - 027 = 640)

43. ¿Qué comando se utiliza para cambiar el grupo propietario de un archivo?

  • A) chown
  • B) chgrp
  • C) chmod
  • D) groupmod
Respuesta correcta: B.

44. ¿Qué hace el comando chown -R user:group /dir?

  • A) Cambia el propietario y grupo de /dir y su contenido recursivamente
  • B) Cambia solo el propietario de /dir y su contenido
  • C) Cambia solo el grupo de /dir y su contenido
  • D) Mueve /dir a /dir/user:group

45. ¿Cuál es la ruta absoluta del directorio que contiene los archivos de configuración de systemd?

46. ¿Qué hace la opción -mtime -7 en find?

47. ¿Qué hace la opción -perm /2000 en find?

  • A) Busca archivos con el bit SUID establecido

  • C) Busca archivos con el bit Sticky establecido

48. ¿Cuál es el significado del permiso 1000 (Sticky Bit) en un directorio?

  • A) Solo el propietario puede eliminar o renombrar sus archivos

  • C) Los ejecutables se ejecutan con los permisos del propietario
  • D) Nadie puede eliminar archivos en ese directorio
Respuesta correcta: A. (Comúnmente usado en /tmp)

49. El comando find . -type f -size +100k -and -size -1M muestra:

  • A) Archivos entre 100 KB y 1 MB
  • B) Archivos menores de 100 KB
  • C) Archivos mayores de 1 MB
