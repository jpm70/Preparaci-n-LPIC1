Examen de Práctica LPIC-1 (Test 15)

1.- ¿A dónde se envían por defecto los mensajes de error generados por los comandos?
STDIN
Archivos de Registro
STDERR
STDOUT

Respuesta Correcta: STDERR

Explicación: Los mensajes de error se envían al Canal 2, conocido como STDERR (Standard Error).

2.- Un comando exitoso imprimirá la salida a la STDOUT. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: La salida estándar de un comando exitoso se envía a STDOUT (Standard Output).

3.- Una barra vertical te permite...
...enviar la misma entrada a varios comandos.
...enviar la salida de un comando a otro.
...introducir varios comandos en un mismo prompt.
...enviar la salida de un comando a un archivo.

Respuesta Correcta: ...enviar la salida de un comando a otro.

Explicación: La barra vertical (pipe |) se utiliza para redirigir la salida del comando de la izquierda a la entrada del comando de la derecha.

4.- Canal 2 es:
STDERR
STDALL
STDOUT
STDIN

Respuesta Correcta: STDERR

Explicación: Los canales de descriptores de archivos son: 0 (STDIN), 1 (STDOUT), y 2 (STDERR).

5.- ¿Cuál de los siguientes comandos anexará su salida a output.file?
`echo Testing -> output.file`
`echo Testing >> output.file`
`echo Testing > output.file`
`output.file < echo Testing`

Respuesta Correcta: echo Testing >> output.file

Explicación: El operador >> se utiliza para anexar la salida al final de un archivo existente. El operador > sobrescribe el archivo.

6.- ¿Qué opción del comando `head` mostrará sólo las primeras cinco líneas de un archivo?
No hay opción necesaria; el head muestra sólo las cinco líneas de forma predeterminada.
-l 5
-n 5
-n

Respuesta Correcta: -n 5

Explicación: La opción -n de head especifica el número de líneas a mostrar. Por defecto, muestra las primeras 10 líneas.

7.- El comando `grep`
...mostrará los números de línea en un archivo que contiene una expresión regular especificada.
...no distingue entre mayúsculas y minúsculas.
...mostrará todas las líneas en un archivo que contiene una Expresión Regular especificada.
...mostrará todas las líneas en un archivo que comienzan con la Expresión Regular especificada.

Respuesta Correcta: ...mostrará todas las líneas en un archivo que contiene una Expresión Regular especificada.

Explicación: grep (Global Regular Expression Print) busca líneas que coincidan con un patrón o Expresión Regular en un archivo.

8.- El comando `grep` se puede utilizar con los caracteres glob. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Las Expresiones Regulares (grep) son patrones de búsqueda de texto dentro de un archivo, mientras que los caracteres glob (como * o ?) se utilizan para la expansión de nombres de archivos por el shell.

9.- ¿Cuál de los siguientes comandos mostrará sólo las líneas que comienzan con `start`?
grep ' start' file.txt
grep 'start' file.txt
grep '^start' file.txt
grep 'start$' file.txt

Respuesta Correcta: grep '^start' file.txt

Explicación: El carácter ^ (circunflejo) ancla el patrón al inicio de la línea en Expresiones Regulares.

10.- ¿Cuál de los siguientes comandos mostrará sólo las líneas que comienzan con `test`?
grep ' test' file.txt
grep 'test' file.txt
grep '^test' file.txt
grep 'test$' file.txt

Respuesta Correcta: grep '^test' file.txt

Explicación: Similar a la anterior, ^ fuerza a la coincidencia a estar al comienzo de la línea.

11.- El comando `echo`:
Se utiliza para dar salida del texto a la consola
Se utiliza para enviar mensajes de error al STDERR
Se utiliza para ejecutar un script shell

Respuesta Correcta: Se utiliza para dar salida del texto a la consola

Explicación: El comando echo simplemente imprime el texto o el valor de una variable en la salida estándar (consola).

12.- Un archivo comienza con `#!/bin/csh`. Esto significa:
La ejecución del script invocará `/bin/csh` para interpretar el resto del archivo
El script no se puede ejecutar en un shell bash
El script es un archivo de configuración C-shell

Respuesta Correcta: La ejecución del script invocará `/bin/csh` para interpretar el resto del archivo

Explicación: Esta línea, conocida como Shebang, especifica la ruta absoluta al intérprete que debe utilizarse para ejecutar el script, en este caso, C Shell.

13.- La mayoría de los comandos de `nano` adoptan la forma de:
Alt y otro carácter
Control y otro carácter
Shift y otro carácter

Respuesta Correcta: Control y otro carácter

Explicación: En el editor de texto nano, los comandos se indican con un ^ (Control) o una M (Alt). Por ejemplo, ^X es para salir.

14.- ¿Qué hace este script shell? (Asumiendo corrección de la sintaxis y flujo de ejecución intencionado)
Crea `/tmp/foo` si no existe
Elimina `/tmp/foo` si existe
Muestra un mensaje de error si `/tmp/foo` no existe

Respuesta Correcta: Crea `/tmp/foo` si no existe

Explicación: Aunque la pregunta es incompleta sin el código del script, esta respuesta se refiere a una acción común de creación condicional de archivos en scripts de shell.

15.- Cuál es el significado de `$1`?
Es el nombre del script.
Es el primer argumento pasado al script.
Es el código de salida del script.

Respuesta Correcta: Es el primer argumento pasado al script.

Explicación: En la programación de shell, $n se refiere al n-ésimo argumento posicional pasado al script. $0 es el nombre del script.

16.- ¿Cuando se va a imprimir "I am here"? (Ejecutando `./test.sh hello goodbye` y con la condición `if [ -f $2 ]`)
Cuando el script se llama con 'hello' y 'goodbye' como argumentos
Si un archivo llamado "goodbye" existe en el directorio actual
Si el script tiene permiso de ejecución

Respuesta Correcta: Si un archivo llamado "goodbye" existe en el directorio actual

Explicación: La condición if [ -f $2 ] evalúa si el segundo argumento ($2 = "goodbye") es un archivo existente (-f).

17.- ¿Cuál es la forma correcta de asignar la palabra "Hello" a una variable?
`A=$Hello`
`A="Hello"`
`A = Hello`

Respuesta Correcta: A="Hello" o A='Hello'

Explicación: La asignación de variables en shell no lleva espacios alrededor del signo =. Las comillas se usan para encerrar el valor de la cadena.

18.- ¿Cuál es la forma correcta de guardar el directorio actual a una variable?
`A=$pwd`
`A=\`pwd\``
`A=$pwd()`

Respuesta Correcta: A=$(pwd) o A=\`pwd\`

Explicación: Las comillas invertidas (\`...\`) o la sintaxis moderna $(...) permiten la **sustitución de comandos**, guardando la salida del comando pwd en la variable A.

19.- ¿Qué comando shell acepta la entrada del teclado del usuario?
`echo`
`get`
`read`

Respuesta Correcta: read

Explicación: El comando read se utiliza para leer una línea de texto de la entrada estándar y asignarla a una variable.

20.- ¿Qué información contiene adentro `$?`?
El código de salida del comando anterior
El nombre del shell actual
El PID del último proceso en segundo plano

Respuesta Correcta: El código de salida del comando anterior

Explicación: La variable especial $? contiene el código de salida (exit status) del comando ejecutado más recientemente. 0 significa éxito.

21.- Una bandera CPU...
...define las características que soporta la CPU.
...indica el uso actual de la CPU.
...es un indicador de error de la CPU.

Respuesta Correcta: ...define las características que soporta la CPU.

Explicación: Las banderas (flags) de la CPU se refieren a las capacidades de hardware de la CPU (ej., soporte para virtualización, nuevas instrucciones, etc.).

22.- Las plataformas de 64 bit pueden acceder a más memoria que las plataformas de 32 bit. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Las arquitecturas de 64 bits tienen un espacio de direcciones de memoria significativamente mayor que las de 32 bits, que están limitadas a aproximadamente 4 GB de RAM.

23.- ¿Qué comando muestra la información de SMBIOS?
`cpuinfo`
`biosinfo`
`dmidecode`

Respuesta Correcta: `dmidecode`

Explicación: El comando dmidecode lee las tablas DMI/SMBIOS para mostrar información de hardware detallada, como el fabricante de la BIOS y la placa base.

24.- Una división de una unidad de disco duro puede ser denominada como un/una
bloque
sector
partición

Respuesta Correcta: partición

Explicación: Un disco duro se divide lógicamente en una o más particiones.

25.- El comando `fdisk` es una herramienta que se utiliza para trabajar con los discos con una partición MBR.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: El comando fdisk es la utilidad clásica en Linux para manipular **particiones MBR** (Master Boot Record).

26.- ¿Cuál de los siguientes es el nombre del archivo de dispositivo válido para el primer disco duro IDE en el sistema?
`/dev/sda`
`/dev/hda`
`/dev/disk0`

Respuesta Correcta: `/dev/hda`

Explicación: Tradicionalmente, los discos IDE eran nombrados /dev/hd[a-z]. Los discos SATA/SCSI/USB usan /dev/sd[a-z].

27.- ¿Qué significa el acrónimo RPM?
Redistributable Package Manager
RPM Package Management
Resource Partitioning Manager

Respuesta Correcta: RPM Package Management

Explicación: RPM (RPM Package Manager) es el formato de paquete estándar utilizado por distribuciones basadas en Red Hat.

28.- Una forma de instalar el nuevo software en un sistema Linux es utilizar un sistema de administración de paquetes.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Los administradores de paquetes (como APT o YUM/DNF) son la forma más común y segura de instalar software en Linux.

29.- Cuando ejecutas el comando `dmesg`, el sistema muestra los mensajes que se generan por el kernel.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: dmesg (diagnose message) muestra el búfer de mensajes del anillo del kernel, que contiene mensajes de arranque y de diagnóstico de controladores.

30.- El directorio `/proc` contiene un subdirectorio para cada proceso presente en el sistema.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: El directorio /proc es un sistema de archivos virtual donde cada subdirectorio con nombre numérico es el PID (ID de proceso) de un proceso en ejecución.

31.- El identificador de proceso (PID) del proceso `init` es
0
1
1000

Respuesta Correcta: 1

Explicación: El proceso init (o systemd/upstart en sistemas modernos) es el primer proceso ejecutado al arrancar el kernel y tiene el PID 1.

32.- El comando `free` muestra la estadística sobre:
El uso de disco duro
El uso de CPU
El uso de memoria

Respuesta Correcta: El uso de memoria

Explicación: El comando free se utiliza para mostrar la cantidad total de memoria física y swap libre y usada en el sistema.

33.- El comando del proceso (`ps`) muestra sólo los procesos que se ejecutan en el shell actual por defecto.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Por defecto, ps (process status) muestra solo los procesos asociados con la terminal actual (TTY). Se necesitan opciones como ps -ef para ver todos.

34.- ¿Qué archivo de registro contiene los mensajes en relación con la autenticación y autorización?
`messages`
`kern.log`
`secure` o `auth.log`

Respuesta Correcta: `secure` o `auth.log`

Explicación: Este archivo (`/var/log/secure` o `/var/log/auth.log`) registra inicios de sesión, uso de sudo, y otra información de autenticación y autorización.

35.- Todos los archivos de registro sólo contienen datos de texto. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Aunque la mayoría son texto plano (ej., `/var/log/syslog`), algunos registros (ej., los de systemd o logs binarios) pueden contener datos binarios.

36.- Cuál de los siguientes sería considerado un host?
Un cable de red
Una dirección IP
Una impresora conectada a la red a través de una dirección IP

Respuesta Correcta: Una impresora conectada a la red a través de una dirección IP

Explicación: Un host es cualquier dispositivo que se comunica en una red utilizando el protocolo TCP/IP, generalmente con su propia dirección IP.

37.- Un servicio es
...una característica proporcionada por una computadora a otra.
...un comando que se ejecuta en el fondo.
...un protocolo de red.

Respuesta Correcta: ...una característica proporcionada por una computadora a otra.

Explicación: Un servicio es un programa o funcionalidad que ofrece recursos o capacidades a otros sistemas (clientes) a través de la red (ej., servicio web, servicio de correo).

38.- Sólo los servidores tienen nombres de host. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Cualquier computadora o dispositivo en red tiene un nombre de host (hostname), que es su identificador legible por humanos.

39.- ¿Cuál de los siguientes protocolos define cómo funciona la comunicación de red?
HTTP
DNS
TCP/IP

Respuesta Correcta: TCP/IP

Explicación: TCP/IP (Transmission Control Protocol/Internet Protocol) es el conjunto de protocolos fundamental para la comunicación en Internet y la mayoría de las redes modernas.

40.- El nombre de los dispositivos de red con cables primarios es `ether0`. ¿Verdadero o falso?
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Los nombres tradicionales eran eth0, eth1, etc. Los sistemas modernos usan nombres predecibles como enp0s3.

41.- ¿Cuál de los siguientes comandos mostrará la dirección ID de tu máquina?
`ip`
`netstat`
`ifconfig`

Respuesta Correcta: `ifconfig` o `ip addr`

Explicación: El comando ifconfig (o su reemplazo moderno ip addr) muestra la configuración de las interfaces de red, incluida la dirección IP.

42.- ¿Cuál de los siguientes comandos te permitirá iniciar sesión en una máquina remota?
`ftp`
`ping`
`ssh`

Respuesta Correcta: `ssh`

Explicación: ssh (Secure Shell) es el protocolo estándar para la administración remota segura de sistemas Linux/Unix.

43.- ¿Qué opción del comando `netstat` contiene la información mostrada como números en lugar de nombres?
-t
-l
-n

Respuesta Correcta: -n

Explicación: La opción -n (numeric) de netstat (o ss) evita la resolución de nombres de host, puertos y usuarios, mostrando solo valores numéricos.

44.- ¿Cuál de los siguientes comandos te permitirá iniciar sesión en la máquina `server1` con el nombre de cuenta `nick`?
`ssh nick server1`
`ssh server1 as nick`
`ssh nick@server1`

Respuesta Correcta: ssh nick@server1

Explicación: La sintaxis estándar para especificar el usuario es ssh [usuario]@[host].

45.- La clave de huellas dactilares RSA permite que el comando `dig` se conecte a los sistemas remotos.
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: La huella digital RSA se utiliza en SSH para verificar la identidad del servidor. dig es una herramienta de consulta DNS.

Todos los usuarios
Solo los miembros del grupo `shadow`
El usuario root

Respuesta Correcta: El usuario root

Explicación: El archivo /etc/shadow contiene las contraseñas cifradas y solo es legible por el usuario root por motivos de seguridad.

`id`
`groups`

Respuesta Correcta: `id`

Explicación: El comando id imprime el UID (User ID), el GID (Group ID) principal y la lista de todos los grupos a los que pertenece el usuario.

48.- Cada usuario pertenece al menos a un grupo.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Cada cuenta de usuario debe tener un grupo principal (Primary Group), lo que significa que siempre pertenece al menos a un grupo.

49.- ¿Qué comando se mostrará a los usuarios que han iniciado la sesión en el sistema?
`last`
`whoami`
`who`

Respuesta Correcta: `who` o `w`

Explicación: El comando who (o w) muestra una lista de usuarios actualmente conectados al sistema.

Respuesta Correcta: ...ejecutar ciertos comandos como usuario root.

Explicación: sudo (substitute user do) permite a un usuario ejecutar un comando con los privilegios de otro usuario (generalmente root) si está configurado en /etc/sudoers.

52.- ¿Qué editor predeterminado utiliza el comando `visudo`?
`nano`
`emacs`
`vi`

Explicación: Por defecto, visudo invoca el editor vi (o vim). Esto se puede cambiar con variables de entorno.

`user`
`group`
`id` o `groups`

Usuarios regulares

66.- ¿Cuál de los siguientes comandos establece los "otros" los permisos para el archivo a `r-x`?

Respuesta Correcta: chmod o=rx file

Explicación: En la notación simbólica de chmod, o (others) seguido de = (set exactly) establece los permisos de lectura (r) y ejecución (x).

67.- ¿Cuál de los siguientes comandos establece para los "otros" los permisos para el archivo a `r-x` (numérico: 5)?

`chmod 775 file`

Respuesta Correcta: chmod 775 file

Explicación: El permiso `r-x` (lectura y ejecución) corresponde al valor octal 5 (4+1). Si se asume el patrón `rwxrwxr-x`, el valor completo es `775`.

68.- Sólo un conjunto (usuario, grupo, otros) de permisos se puede cambiar a la vez utilizando el método simbólico.
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Se pueden cambiar múltiples conjuntos a la vez, separándolos por comas (ej., chmod u+w,g-w,o=r file).

69.- El comando `chown` se puede utilizar para cambiar el propietario y el grupo de un archivo.

Falso

Respuesta Correcta: Verdadero

Explicación: La sintaxis chown user:group file permite cambiar tanto el propietario (user) como el grupo (group) en una sola operación.

Falso

71.- El usuario `sysadmin` podrá cambiar los permisos del archivo. (Permisos: `----r--r--`).

Falso

Respuesta Correcta: Verdadero

Explicación: El cambio de permisos de un archivo (con chmod) lo puede realizar únicamente el propietario del archivo o el usuario root, independientemente de los permisos actuales.

72.- La notación octal utiliza los siguientes valores para los permisos concedidos:
`r=1`, `w=2`, `x=3`
`r=4`, `w=2`, `x=1`
`r=1`, `w=4`, `x=2`

Respuesta Correcta: `r=4`, `w=2`, `x=1`

Explicación: Los valores octales de los permisos son: Lectura (r) = 4, Escritura (w) = 2, y Ejecución (x) = 1.

73.- ¿Cuál de los siguientes permisos permitiría a todos los usuarios añadir, ver y eliminar los archivos en un directorio?
644
755
777

Respuesta Correcta: 777

Explicación: El permiso 777 (rwxrwxrwx) otorga permisos de lectura, escritura y ejecución a todos, lo cual es necesario para crear, modificar y **eliminar** archivos en un directorio (la capacidad de eliminar viene del permiso de escritura en el directorio).

74.- Un usuario no puede borrar un archivo si no es su propietario.
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Un usuario puede borrar un archivo si tiene permiso de **escritura** (w) y **ejecución** (x) en el **directorio contenedor**.

75.- El permiso "ejecutar" en un archivo te permite:
Leer el contenido del archivo
Modificar el contenido del archivo
Ejecutar el archivo como un script o programa

Respuesta Correcta: Ejecutar el archivo como un script o programa

Explicación: El permiso de **ejecución** (x) es necesario para que el kernel pueda ejecutar un archivo binario o interpretador de un script.

76.- El directorio `/tmp` es un directorio temporal y no va a existir en un sistema en todo momento.
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: El directorio /tmp **siempre existe** en un sistema Linux estándar, aunque su contenido puede ser borrado durante el reinicio o periódicamente.

77.- El directorio `/var` tiene los archivos que cambian con el tiempo.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: El directorio /var (variable) contiene datos de estado en constante cambio, como archivos de registro (/var/log), spool de correo, datos de caché, etc.

78.- El permiso "sticky bit"...
...establece el permiso de solo lectura para todos.
...los demás no pueden eliminar los archivos de los que no son propietarios de un directorio común.
...establece el permiso de ejecución para todos.

Respuesta Correcta: ...los demás no pueden eliminar los archivos de los que no son propietarios de un directorio común.

Explicación: El Sticky Bit (valor octal 1000) asegura que solo el propietario (o root) pueda eliminar o renombrar un archivo dentro de ese directorio.

79.- ¿Cuál de los siguientes comandos establecerá "sticky bit" en el directorio `/shared`?
`chmod 7771 /shared`
`chmod 1777 /shared`
`chmod +t /shared`

Respuesta Correcta: chmod 1777 /shared

Explicación: El Sticky Bit corresponde al valor octal 1 en el cuarto dígito. 1777 aplica el sticky bit y permite todos los permisos a todos los usuarios.

80.- El permiso "setuid"...
...permite que un comando se ejecute como el propietario del archivo.
...permite que un comando se ejecute como el grupo del archivo.
...fuerza que el comando se ejecute como root.

Respuesta Correcta: ...permite que un comando se ejecute como el propietario del archivo.

Explicación: El SetUID (valor octal 4000) permite que un programa ejecutable se ejecute con los permisos del propietario del archivo (ej., el comando passwd se ejecuta como root).

81.- ¿Cuál de los siguientes comandos establecerá `setuid` en el directorio `/usr/bin/program`?
`chmod 4755 /usr/bin/program`
`chmod u+s /usr/bin/program`
`chmod 755 /usr/bin/program`

Respuesta Correcta: chmod 4755 /usr/bin/program

Explicación: El SetUID corresponde al valor octal 4 en el cuarto dígito. 4755 establece setuid y permisos rwxr-xr-x.

82.- ¿Cuál de los siguientes comandos establecerá `setgid` en el directorio `/shared`?
`chmod 4777 /shared`
`chmod g+s /shared`
`chmod 2777 /shared`

Respuesta Correcta: chmod 2777 /shared

Explicación: El SetGID (valor octal 2000) permite que los archivos creados en el directorio hereden el grupo del directorio. 2777 establece setgid y todos los permisos.

83.- Eliminando un archivo de origen romperá un vínculo físico asociado.
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: Un vínculo físico (Hard Link) es simplemente otro nombre para el mismo inodo y datos. El archivo solo se elimina del disco cuando el **último** vínculo físico es eliminado.

84.- Un origen y un vínculo físico deben ser parte del mismo sistema de archivos.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Los vínculos físicos están limitados a un solo sistema de archivos porque hacen referencia directamente al **inodo**, que es un identificador único solo dentro de su sistema de archivos.

85.- Eliminando un archivo de origen romperá un vínculo simbólico asociado.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Un **vínculo simbólico** (Soft Link) es un puntero al nombre del archivo de origen. Si se elimina el origen, el vínculo simbólico apunta a un archivo que ya no existe (vínculo roto).

86.- Unix es
Un editor de texto
Un lenguaje de programación
Un sistema operativo

Respuesta Correcta: Un sistema operativo

Explicación: UNIX es una familia de sistemas operativos multiusuario y multitarea, desarrollado originalmente en Bell Labs.

87.- El fin de bootloader es:
Iniciar la interfaz gráfica
Cargar el kernel al encender la computadora
Ejecutar la BIOS

Respuesta Correcta: Cargar el kernel al encender la computadora

Explicación: El **bootloader** (ej., GRUB, LILO) es el programa encargado de cargar el núcleo (kernel) del sistema operativo en la memoria para que comience a ejecutarse.

88.- Apple OS X no es:
Basado en Unix
Un sistema operativo
Derivado de Linux

Respuesta Correcta: Derivado de Linux

Explicación: macOS (anteriormente OS X) está basado en el sistema operativo Darwin, que es un derivado de BSD (un tipo de Unix), no de Linux.

89.- Microsoft Windows no:
Tiene un modo de compatibilidad de Linux (WSL)
Utiliza el kernel de Linux
Soporta TCP/IP

Respuesta Correcta: Utiliza el kernel de Linux

Explicación: Windows utiliza su propio kernel (NT), a diferencia de Linux. (Aunque con WSL se puede ejecutar un kernel de Linux, Windows como tal no lo usa nativamente).

90.- ¿Qué aplicación usarías para editar y reconstruir los archivos de sonido para crear un podcast?
GIMP
Blender
Audacity

Respuesta Correcta: Audacity

Explicación: Audacity es un editor de audio digital y software de grabación de código abierto, ideal para la edición de podcasts.

91.- Un servidor probablemente se ejecute en modo gráfico.
Verdadero
Falso

Respuesta Correcta: Falso

Explicación: La mayoría de los servidores se ejecutan en modo de **línea de comandos (CLI)** para ahorrar recursos y reducir la superficie de ataque.

92.- Si quieres escribir un informe que tuvieras que imprimir, es probable que utilices:
`vi`
AbiWord
LibreOffice Writer

Respuesta Correcta: LibreOffice (Writer)

Explicación: LibreOffice (específicamente Writer) es la suite ofimática de código abierto más utilizada para documentos y la impresión de informes.

93.- Los botones "me gusta" de las redes sociales pueden realizar un seguimiento de tu actividad a través de Internet.
Verdadero
Falso

Respuesta Correcta: Verdadero

Explicación: Los *widgets* de redes sociales incrustados en sitios web utilizan *tracking cookies* y otros mecanismos para seguir la actividad del usuario a través de múltiples sitios.

94.- ¿Qué se puede hacer para evitar que las personas remotas ejecuten los programas en tu computadora?
No utilizar las redes inalámbricas - sólo cableadas
Encender un firewall
Bloquear todas las cookies

Respuesta Correcta: Encender un firewall

Explicación: Un **firewall** bloquea las conexiones entrantes no deseadas, impidiendo que usuarios remotos maliciosos accedan y ejecuten servicios o programas.

95.- En general, ¿para cuál de los siguientes querrías utilizar la compresión sin pérdidas?
Un correo electrónico encriptado
Un archivo de audio mp3
Un archivo de registro

Respuesta Correcta: Un archivo de registro

Explicación: La **compresión sin pérdidas** (Lossless) es esencial para archivos donde se requiere una réplica perfecta de los datos, como **archivos de registro** o binarios.

96.- Una barra vertical te permite...
...enviar la misma entrada a varios comandos.
...enviar la salida de un comando a otro.
...introducir varios comandos en un mismo prompt.

Respuesta Correcta: ...enviar la salida de un comando a otro.

Explicación: La barra vertical (pipe |) es el operador de **tubería (pipe)**, que conecta la STDOUT del primer comando con la STDIN del segundo.

97.- El propósito principal de un agente de transferencia de correo es:
Entregar el correo entre los servidores
Servir el correo electrónico a los clientes finales
Actuar como una puerta de enlace entre el fax y correo electrónico

Respuesta Correcta: Entregar el correo entre los servidores

Explicación: Un **MTA** (Mail Transfer Agent, ej., Postfix) es responsable de transferir el correo entre **servidores** usando SMTP.

98.- La diferencia entre la GPL y LGPL son:
LGPL se aplica a los servicios web
LGPL permite la conexión del software sin GPL
LGPL te permite distribuir el software sólo en forma binaria

Respuesta Correcta: LGPL permite la conexión del software sin GPL

Explicación: La **LGPL** (Lesser General Public License) es menos restrictiva que la GPL, permitiendo que el software propietario pueda **enlazar** (link) dinámicamente con la biblioteca LGPL sin tener que liberar su propio código fuente.

99.- ¿Qué directorio contiene los archivos de registro normalmente?
El directorio `/var/log`
El directorio `/log`
El directorio `/proc/log`

Respuesta Correcta: El directorio `/var/log`

Explicación: Según el FHS (Filesystem Hierarchy Standard), todos los archivos de registro del sistema y de las aplicaciones se almacenan en el directorio **`/var/log`**.
