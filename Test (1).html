1. ¿Cómo podría crear una copia de seguridad del sistema usando el comando "dd"?

  • a) dd if=/dev/sda of=/media/usb_stick bs=4M count=$(blocksize -h) conv=noerror,sync

  • c) tar cvfz backup.tar.gz --exclude='/proc' --one-file-system /

La respuesta correcta es: a (para copiar el sistema completo, incluyendo particiones y sistemas de archivos).
